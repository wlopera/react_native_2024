### LEVANTAR APP
* > npm cache clean --force
* > npm cache verify
* > npm -g uninstall expo-cli --save
* > npm install expo-cli --global
* > expo init guess-game
* > cd app-name
* > npm start

* limpiar cache: > npm cache clean --force
